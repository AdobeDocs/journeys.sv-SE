# trim {#trim}

Tar bort start- och slutblanksteg.

## Kategori

Sträng

## Funktionssyntax

`trim(<parameters>)`

## Parameter

| Parameter | Typ |
|-----------|------------------|
| string | string |

## Signatur och returtyp

`trim(<string>)`

Returnera en sträng.

## Exempel

`trim(" Hello ")`

Returnerar &quot;Hello&quot;.
