# uuid {#uuid}

Skapar ett slumpmässigt UUID (Unikt IDentifier).

## Kategori

Sträng

## Funktionssyntax

`uuid()`

## Parametrar

Den här funktionen kräver inga parametrar.

## Signatur och returtyp

`uuid()`

Returnerar en sträng.

`uuid()`
