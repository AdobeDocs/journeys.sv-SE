# isEmpty {#isEmpty}

Returnerar true om strängen i parametern inte har något tecken.

## Kategori

Sträng

## Funktionssyntax

`isEmpty(<parameters>)`

## Parametrar

* string

## Signatur och returtyp

`isEmpty(<string>)`

Returnerar ett booleskt värde.

## Exempel

`isEmpty("")`

Returnerar true.

`isEmpty("Hello World")`

Returnerar false.
