# isNotEmpty {#isNotEmpty}

Returnerar true om strängen i parametern inte är tom.

## Kategori

Sträng

## Funktionssyntax

`isNotEmpty(<parameters>)`

## Parametrar

* string

## Signatur och returtyp

`isNotEmpty(<string>)`

Returnerar ett booleskt värde.

## Exempel

`isNotEmpty("")`

Returnerar false.

`isNotEmpty("hello")`

Returnerar true.
